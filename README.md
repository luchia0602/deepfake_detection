# Deepfake Detection

Benchmarking audio deepfake detection features on Portuguese speech. Compares handcrafted and deep representations (MFCC, CQCC, Wav2Vec2, HuBERT, Whisper, etc.) for detecting synthetic speech generated by modern neural TTS systems.

---

# User Manual

## 1. Requirements

```bash
pip install datasets librosa numpy pandas scikit-learn tqdm scipy \
    python_speech_features amfm_decompy transformers torch
```

The CQCC extractor also requires the ASVspoof 2021 baseline folder to be present at:

```text
2021/LA/Baseline-CQCC-GMM/python/
```

relative to the project root (clone it from the ASVspoof 2021 repository separately).

---

## 2. Pipeline Overview

The four scripts must be run in order. Each step produces files consumed by the next one.

```text
prepare_splits.py
        ↓
extract_features.py
        ↓
evaluate.py
        ↓
analyse.py
```

---

## 2.1 Build Split CSVs

### Command

```bash
python scripts/prepare_splits.py [-h] [--dataset-id DATASET_ID]
                                 [--output-dir OUTPUT_DIR]
                                 [--train-ratio TRAIN_RATIO]
                                 [--val-ratio VAL_RATIO]
                                 [--seed SEED]
```

### Description

Streams metadata from HuggingFace, builds balanced train / val / test splits (equal bonafide / spoof counts, spoof diversity-aware), and writes `train.csv`, `val.csv`, and `test.csv` to `--output-dir`.

### Arguments

| Argument        | Default                      | Description                                     |
| --------------- | ---------------------------- | ----------------------------------------------- |
| `--dataset-id`  | `AKCIT-Deepfake/BRSpeech-DF` | HuggingFace dataset repo ID                     |
| `--output-dir`  | `artifacts/`                 | Where to write the three CSVs                   |
| `--train-ratio` | `0.8`                        | Fraction of bonafide examples used for training |
| `--val-ratio`   | `0.1`                        | Fraction used for validation                    |
| `--seed`        | `42`                         | Random seed for reproducibility                 |

### Example

```bash
python scripts/prepare_splits.py
```

### Output

```text
artifacts/
├── train.csv
├── val.csv
└── test.csv
```

## 2.2 Extract Features

### Command

```bash
python scripts/extract_features.py [-h] \
    --representation {prosodic,whisper,xlsr,mfcc,lfcc,cqcc} \
    [--features-dir FEATURES_DIR] \
    [--dataset-id DATASET_ID] \
    [--meta-dir META_DIR] \
    [--force]
```

### Description

Streams audio from HuggingFace for each split in the CSVs, runs the chosen extractor, and saves `X_<name>.npy` and `y.npy` arrays.

Run once per representation.

### Arguments

| Argument           | Default                      | Description                     |
| ------------------ | ---------------------------- | ------------------------------- |
| `--representation` | required                     | Feature type to extract         |
| `--features-dir`   | `artifacts/features/`        | Where to write feature arrays   |
| `--dataset-id`     | `AKCIT-Deepfake/BRSpeech-DF` | HuggingFace dataset repo ID     |
| `--meta-dir`       | `artifacts/`                 | Directory containing split CSVs |
| `--force`          | off                          | Overwrite existing files        |

### Example (single representation)

```bash
python scripts/extract_features.py --representation prosodic
```

### Example (all representations)

```bash
for rep in prosodic mfcc lfcc cqcc xlsr whisper
do
    python scripts/extract_features.py --representation $rep
done
```

### Output

```text
artifacts/features/

train/
├── X_prosodic.npy
└── y.npy

val/
├── X_prosodic.npy
└── y.npy

test/
├── X_prosodic.npy
└── y.npy
```

## 2.3 Train and Evaluate

### Command

```bash
python scripts/evaluate.py [-h] \
    --representation REP [REP ...] \
    [--features-dir FEATURES_DIR] \
    [--meta-dir META_DIR] \
    [--force]
```

### Description

Trains a Logistic Regression classifier for each representation, evaluates it on the test set, and saves predictions.

Representations whose predictions already exist are skipped unless `--force` is specified.

### Arguments

| Argument           | Default               | Description                     |
| ------------------ | --------------------- | ------------------------------- |
| `--representation` | required              | One or more feature types       |
| `--features-dir`   | `artifacts/features/` | Root features directory         |
| `--meta-dir`       | `artifacts/`          | Directory containing split CSVs |
| `--force`          | off                   | Overwrite existing predictions  |

### Example (single representation)

```bash
python scripts/evaluate.py --representation prosodic
```

### Example (multiple representations)

```bash
python scripts/evaluate.py \
    --representation prosodic mfcc lfcc cqcc xlsr whisper
```

### Output

```text
artifacts/features/predictions/

prosodic/
├── probs_test.npy
├── preds_test.npy
└── metrics.json
```

## 2.4 Analyse and Plot

### Command

```bash
python scripts/analyse.py [-h] \
    --representation REP [REP ...] \
    [--features-dir FEATURES_DIR] \
    [--meta-dir META_DIR] \
    [--save-plots] \
    [--plots-dir PLOTS_DIR] \
    [--skip-pca]
```

### Description

Loads saved predictions, prints a metrics table and error analysis, and generates six plots.

Requires `evaluate.py` to have been run first.

### Arguments

| Argument           | Default               | Description                     |
| ------------------ | --------------------- | ------------------------------- |
| `--representation` | required              | One or more feature types       |
| `--features-dir`   | `artifacts/features/` | Root features directory         |
| `--meta-dir`       | `artifacts/`          | Directory containing `test.csv` |
| `--save-plots`     | off                   | Save plots to disk              |
| `--plots-dir`      | `results/plots/`      | Plot output directory           |
| `--skip-pca`       | off                   | Skip PCA visualisations         |

### Example (interactive)

```bash
python scripts/analyse.py --representation prosodic
```

### Example (save plots)

```bash
python scripts/analyse.py \
    --representation prosodic mfcc lfcc cqcc xlsr whisper \
    --save-plots \
    --plots-dir results/plots
```

### Example (skip PCA)

```bash
python scripts/analyse.py \
    --representation xlsr whisper \
    --save-plots \
    --skip-pca
```

### Plots Produced

```text
results/plots/
├── roc_curves.png
├── confusion_matrices.png
├── per_model_accuracy.png
├── error_heatmap.png
├── pca_by_label.png
└── pca_by_model.png
```
