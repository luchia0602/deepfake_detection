# deepfake_detection
Benchmarking audio deepfake detection features on Portuguese speech. Compares handcrafted and deep representations (MFCC, CQCC, Wav2Vec2, HuBERT, Whisper, etc.) for detecting synthetic speech generated by modern neural TTS systems.


## User Manual

  # 1.	Requirements
  
        pip install datasets librosa numpy pandas scikit-learn tqdm scipy \
            python_speech_features amfm_decompy transformers torch
The CQCC extractor also requires the ASVspoof 2021 baseline folder to be present at 2021/LA/Baseline-CQCC-GMM/python/ relative to the project root (clone it from the ASVspoof 2021 repository separately).

  # 2.	Pipeline overview

The four scripts must be run in order. Each step produces files consumed by the next one.
prepare_splits.py  →  extract_features.py  →  evaluate.py  →  analyse.py

    # 2.1 Build split CSVs
          python scripts/prepare_splits.py [-h] [--dataset-id DATASET_ID]
                                  [--output-dir OUTPUT_DIR]
                                  [--train-ratio TRAIN_RATIO]
                                  [--val-ratio VAL_RATIO]
                                  [--seed SEED]
                                  
Streams metadata from HuggingFace, builds balanced train / val / test splits (equal bonafide / spoof counts, spoof diversity-aware), and writes train.csv, val.csv, test.csv to --output-dir.

Argument	                      Default	                    Description
--dataset-id                    AKCIT-Deepfake/BRSpeech-DF	HuggingFace dataset repo ID
--output-dir	                  artifacts/	                Where to write the three CSVs
--train-ratio	                  0.8	                        Fraction of bonafide examples used for training
--val-ratio	                    0.1	                        Fraction used for validation (remainder → test)
--seed	                        42	                        Random seed for reproducibility

Example: python scripts/prepare_splits.py
Output:
artifacts/train.csv
artifacts/val.csv
artifacts/test.csv

    # 2.2 Extract features
          python scripts/extract_features.py [-h] --representation {prosodic,whisper,xlsr,mfcc,lfcc,cqcc}
                                    [--features-dir FEATURES_DIR]
                                    [--dataset-id DATASET_ID]
                                    [--meta-dir META_DIR]
                                    [--force]
                                    
Streams audio from HuggingFace for each split in the CSVs, runs the chosen extractor, and saves X_<name>.npy and y.npy arrays. Run once per representation.

Argument	                      Default	                      Description
--representation	              (required)	                  Feature type to extract
--features-dir	                artifacts/features/	          Where to write .npy arrays
--dataset-id	                  AKCIT-Deepfake/BRSpeech-DF	  HuggingFace dataset repo ID
--meta-dir	                    artifacts/	                  Directory containing the split CSVs
--force	                        off	                          Overwrite existing files

Example (single representation): python scripts/extract_features.py --representation prosodic
Example (all representations):
  for rep in prosodic mfcc lfcc cqcc xlsr whisper; do
    python scripts/extract_features.py --representation $rep
  done
Output (per split, per representation):
artifacts/features/train/X_prosodic.npy
artifacts/features/train/y.npy
artifacts/features/val/X_prosodic.npy
artifacts/features/val/y.npy
artifacts/features/test/X_prosodic.npy
artifacts/features/test/y.npy

    # 2.3 Train and evaluate
          python scripts/evaluate.py [-h] --representation REP [REP ...]
                            [--features-dir FEATURES_DIR]
                            [--meta-dir META_DIR]
                            [--force]
                            
Trains a logistic regression classifier for each representation, evaluates it on the test set, and saves predictions. Skips representations whose predictions already exist unless --force is given.

Argument	                      Default	                      Description
--representation	              (required)	                  One or more feature types
--features-dir	                artifacts/features/	          Root features directory
--meta-dir	                    artifacts/	                  Directory containing split CSVs
--force	                        off	                          Overwrite existing predictions

Example (single representation): python scripts/evaluate.py --representation prosodic
Example (multiple representations, summary table printed at the end): python scripts/evaluate.py --representation prosodic mfcc lfcc cqcc xlsr whisper
Output (per representation):
artifacts/features/predictions/prosodic/probs_test.npy
artifacts/features/predictions/prosodic/preds_test.npy
artifacts/features/predictions/prosodic/metrics.json

    # 2.4 Analyse and plot
          python scripts/analyse.py [-h] --representation REP [REP ...]
                           [--features-dir FEATURES_DIR]
                           [--meta-dir META_DIR]
                           [--save-plots]
                           [--plots-dir PLOTS_DIR]
                           [--skip-pca]

Loads saved predictions, prints a metrics table and error analysis, and generates six plots. Requires evaluate.py to have been run first.

Argument	                      Default	                      Description
--representation	              (required)	                  One or more feature types
--features-dir	                artifacts/features/	          Root features directory
--meta-dir	                    artifacts/	                  Directory containing test.csv
--save-plots	                  off	                          Save plots to disk instead of displaying interactively
--plots-dir	                    results/plots/	              Output directory for plot images
--skip-pca	                    off	                          Skip PCA plots (useful for high-dimensional features)

Example (interactive display): python scripts/analyse.py --representation prosodic
Example (save plots, compare all representations): 
python scripts/analyse.py \
    --representation prosodic mfcc lfcc cqcc xlsr whisper \
    --save-plots --plots-dir results/plots
Example: skip PCA (faster for xlsr / whisper)
python scripts/analyse.py --representation xlsr whisper --save-plots --skip-pca
Plots produced
results/plots/roc_curves.png
results/plots/confusion_matrices.png
results/plots/per_model_accuracy.png
results/plots/error_heatmap.png
results/plots/pca_by_label.png        # skipped with --skip-pca
results/plots/pca_by_model.png        # skipped with --skip-pca

    # 2.5 Full example run
    
1. build splits (only needed once): python scripts/prepare_splits.py

2. extract two representations
python scripts/extract_features.py --representation prosodic
python scripts/extract_features.py --representation whisper

3. evaluate both: python scripts/evaluate.py --representation prosodic whisper

4. analyse and save plots: python scripts/analyse.py --representation prosodic whisper --save-plots
