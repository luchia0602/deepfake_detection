# deepfake_detection
Benchmarking audio deepfake detection features on Portuguese speech. Compares handcrafted and deep representations (MFCC, CQCC, Wav2Vec2, HuBERT, Whisper, etc.) for detecting synthetic speech generated by modern neural TTS systems.


## Acknowledgements

- **CQCC code:** adapted from the ASVspoof 2021 Baseline CM & Evaluation Package
  (https://github.com/asvspoof-challenge/2021), specifically the
  `LA/Baseline-CQCC-GMM/python/` directory (which contains the `CQCC/` package).
  Place this directory at the path referenced in the extraction script.
- **Dataset:** BRSpeech-DF (AKCIT-Deepfake/BRSpeech-DF on HuggingFace).
- **Method:** follows Yi et al. (2023), "Audio Deepfake Detection: A Survey."
